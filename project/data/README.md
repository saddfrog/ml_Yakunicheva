# Данные проекта

Датасет содержит 4480 изображений 112 классов таблеток, разбитых на три подвыборки:

| Сплит | Количество изображений | Назначение |
|-------|----------------------|------------|
| **train** | 3136 (70%) | Обучение модели |
| **valid** | 672 (15%) | Валидация во время обучения |
| **test** | 672 (15%) | Финальное тестирование |

**Условия съёмки:**
- Фиксированный масштаб (размер важен для распознавания)
- Однородный фон
- Два режима освещения (верхнее рассеянное и боковое для видимости гравировки)

**Структура данных:**
   - Каждый txt-файл содержит первое число — класс таблетки (0-111), остальные числа — координаты полигона (не использовались).

   ogyeiv2:
      train/
         images/ # 3136 изображений
         labels/ # 3136 txt-файлов с метками классов
      test/
         images/ # 672 изображения
         labels/ # 672 txt-файла
      valid/
      images/ # 672 изображения
      labels/ # 672 txt-файл
      
В репозиторий демонстрационно загружены файлы для проверки работы модели. 

**Важно: **
Данные всего датасета не хранятся в репозитории (добавлены в .gitignore)

В репозиторий демонстрационно загружены файлы для проверки работы модели. 

При первом запуске проекта необходимо скачать датасет по инструкции выше
