# Ноутбуки проекта

В эту папку помещаются **экспериментальные ноутбуки**:

- разведочный анализ данных (EDA);
- прототипы моделей;
- отдельные эксперименты;
- разовые проверки гипотез.

## Ноутбуки в этом проекте

| Ноутбук | Описание | Результат |
|---------|----------|-----------|
| `01_eda_ogyeiv2.ipynb` | Разведочный анализ датасета OGYEIv2: проверка структуры, визуализация примеров, анализ распределения классов | Подтверждена структура 112 классов, ~40 изображений на класс |
| `02_train_compare.ipynb` | Основной эксперимент: обучение и сравнение трёх архитектур (ResNet50, EfficientNet_B0, MobileNet_V3) | Таблица сравнения, графики, выбор лучшей модели |
| `02_train_compare_learning.ipynb` | Сохранённый вывод обучения (для парсинга истории) | Используется для построения графиков без переобучения |


## Связь с `src/`

Весь «боевой» код (Dataset, обучение, API) вынесен в папку `src/`.  
Ноутбуки используются только для **экспериментов и визуализации**.
