# Артефакты проекта

В этой папке хранятся результаты экспериментов и обученные модели.

## Описание артефактов

### Модели (`.pt` файлы)

Сохранённые веса лучших версий каждой архитектуры (по валидационной точности).

| Файл | Модель | Лучшая Val Acc | Размер |
|------|--------|----------------|--------|
| `ResNet50_best.pt` | ResNet50 | 99.70% | ~95 MB |
| `EfficientNet_B0_best.pt` | EfficientNet_B0 | 99.70% | ~20 MB |
| `MobileNet_V3_best.pt` | MobileNet_V3 | 99.26% | ~16 MB |

### Метрики (`final_comparison.csv`)

Сводная таблица результатов тестирования:

| Модель | Best Val Acc | Test Acc | Test F1 |
|--------|--------------|----------|---------|
| ResNet50 | 0.9970 | 0.9985 | 0.9985 |
| EfficientNet_B0 | 0.9970 | 0.9955 | 0.9956 |
| MobileNet_V3 | 0.9926 | 0.9851 | 0.9842 |

### Графики

| Файл | Описание |
